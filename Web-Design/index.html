Summary

The plots on this site show how weather changes in relationship to the distance from the equator. A dataset of over 500 cities pulled from OpenWeatherMap API was used. Matplotlib was used to plot the various aspects of the weather vs. latitude. These included; temperature, cloudiness, hunity and wind speed. This site provides the source data and visualizations created as part of the analysis and an explanation and description of any trends.
